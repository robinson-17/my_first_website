Estrutura básica do HTML.

Voltar

A estrutura básica de uma página HTML inclui as tags <!DOCTYPE html>, <html>, <head>, <title> e <body>.

Falando sobre Tags.

Voltar

Tags são os elementos básicos do HTML. Exemplos incluem: <h1> até <h6> para cabeçalhos, <p> para parágrafos, <mark>, <small>, <i>, <u>, <strong>, <ol>, <ul>, <li>, <a>, <hr>, <sub>, <sup> e <blockquote>.

Atributos básicos das TAG's.

Voltar

Atributos comuns das tags HTML incluem:

  • id: Define um identificador único para o elemento HTML. Pode ser usado para aplicar estilos CSS específicos ou manipular o elemento com JavaScript.
  • style: Especifica estilos CSS inline para o elemento, permitindo a personalização direta de aparência.
  • class: Atribui uma ou mais classes ao elemento, que podem ser utilizadas para aplicar estilos CSS em grupo ou para selecionar elementos com JavaScript.

Atributos específicos incluem:

  • type para <input>: Define o tipo de dado que o campo de entrada deve aceitar (por exemplo, text, password, email, etc.).
  • src para <img>: Especifica o caminho da imagem a ser exibida pelo elemento <img>.

Links.

Voltar

Links em HTML são usados para navegar entre páginas da web, ou para direcionar o usuário para diferentes seções de uma mesma página. A tag <a> é utilizada para criar um link, e o atributo href especifica o destino do link.

Por exemplo, para criar um link para o site da W3Schools, você pode usar o seguinte código:

<a href="https://www.w3schools.com">Visite o W3Schools</a>

Se você quiser que o link abra em uma nova aba, pode adicionar o atributo target="_blank":

<a href="https://www.w3schools.com" target="_blank">Visite o W3Schools</a>

Além de navegar para outras páginas, os links também podem ser usados para direcionar a diferentes seções da mesma página. Isso é feito utilizando âncoras. Primeiro, adicione um atributo id ao elemento que você deseja que seja o destino do link:

<h2 id="seccao-exemplo">Seção de Exemplo</h2>

Em seguida, crie um link que aponta para essa âncora:

<a href="#seccao-exemplo">Ir para a Seção de Exemplo</a>

Quando o usuário clicar no link, ele será levado diretamente para a seção de exemplo na mesma página.

Essas são as funcionalidades básicas para utilizar links em uma página HTML, permitindo criar uma navegação eficiente e intuitiva.Exemplo: Para mais informações, visite a W3Schools.
